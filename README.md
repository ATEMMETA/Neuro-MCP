## Project Overview
A platform for agent orchestration with a Node.js backend and React Native mobile app.

## Tech Stack
- **Backend**: Node.js, Express, TypeScript
- **Mobile**: React Native
- **CLI**: Commander.js
- **Monorepo**: Turborepo
- **Testing**: Jest, Vitest
- **CI/CD**: GitHub Actions (Semantic Release, CodeQL, Snyk, etc.)
